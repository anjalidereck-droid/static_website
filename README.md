# Static Romantic Website

A responsive single-page static website built with HTML, CSS, and JavaScript. The project demonstrates front-end UI design, interactive effects, media integration, and deployment through GitHub Pages.

## Overview

This project is a browser-based static website designed as an interactive one-page experience. It uses a lightweight front-end stack and focuses on visual presentation, user interaction, responsive layout, and simple client-side behavior without any backend dependencies.

## Features

- Responsive one-page layout
- Interactive yes/no entry screen
- Animated floating heart effects
- Smooth content reveal transitions
- Background music support
- Image gallery / memory section
- Custom themed content cards
- GitHub Pages deployment-ready structure

## Deployment

This project is configured for static hosting on GitHub Pages.

### Deployment steps
1. Push the project files to a public GitHub repository.
2. Ensure the main page is named `index.html`.
3. Open **Settings** → **Pages** in the repository.
4. Select **Deploy from a branch**.
5. Choose the `main` branch and the `/ (root)` folder.
6. Save and wait for the site to publish.

## Live Demo

## Notes

- The repository must be public to use GitHub Pages on a standard free GitHub setup.
- Browsers may block autoplay audio until the visitor interacts with the page.
- All assets should remain in their referenced locations for paths to work correctly.
